# ITR-003 - INDEPENDENCE CHARTER

## Objective

Construct one independently originated technical review without an evaluator defining:

- the artifact;
- the technical question;
- the missing layer;
- the test architecture;
- the classification standard;
- the expected qualification outcome.

## Independence rule

The review must originate from an independently observed technical condition.

External feedback may be considered only after:

1. the artifact is selected;
2. the review question is frozen;
3. the carrier is designed;
4. the tests are executed;
5. the initial determination is issued.

## Required sequence

Observation
→ consequence identification
→ artifact selection
→ version freeze
→ review-question freeze
→ original challenge model
→ execution
→ evidence preservation
→ bounded determination
→ external review

## Prohibited sequence

Evaluator criticism
→ evaluator-defined correction
→ implementation of correction
→ return to evaluator for qualification

## Qualification boundary

ITR-003 will not claim general reviewer qualification.

It will demonstrate only the independently originated capability actually supported by the final carrier and evidence.
